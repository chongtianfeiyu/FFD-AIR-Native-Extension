This repository contains code to go along with the article "Writing your first AIR Native Extension" written for Flash & Flex Developer's Magazine.

The native extension created here allows an AIR for iOS app to access the Twitter framework provided by iOS 5, allowing Flash apps to seamlessly integrate with Twitter.
